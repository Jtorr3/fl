# GRIT — sidechained distortion

## What It Is

GRIT is a saturation stage whose character is *shaped by a sidechain signal* rather than a
static knob. Feed the main input the sound to distort and route a sidechain (kick, drum bus,
vocal, …) to GRIT's aux input; the sidechain's envelope in a focus band drives the amount and
shape of the distortion, so the grit pumps and breathes with the groove instead of sitting flat.
It is built for dark-techno rumble, drum-bus heat, and destructive vocal-rip textures.

## Signal Flow

```
main in ─ trim ─ pre-filter(SVF HP/LP) ─┐
                                        ├─ DIST CORE ─ post-filter ─ auto-gain ─ mix ─ out
sidechain in ─ SC focus BP ─ env follower┘        ▲
                                                  └ mode selects how SC drives the core
```

Nonlinear stages run at **4x oversampling** (polyphase halfband, `suite_core::dsp::Oversampler4x`).
Output is hard-ceilinged at ±0.999 (≤ 0 dBFS) as a safety guard.

## Modes

- **A · Env-Drive** — the sidechain envelope raises the drive amount:
  `drive_dB(t) = drive + depth·36dB · env(t)^curve`. More sidechain → more harmonics.
- **B · Waveshape** — the sidechain envelope injects a dynamic bias into the
  waveshaper: `y = shape(x·drive + depth·2·env(t))`. The bias shifts the operating
  point, pumping even-harmonic character with the sidechain. A DC blocker follows.
- **C · Spectral** — *deferred* (see `DEFERRED.md`); not selectable in this build.

## Controls

- **Mode** — how the sidechain drives the core: **A · Env-Drive** (SC envelope raises drive) or
  **B · Waveshape** (SC injects a dynamic bias). Env-Drive for pumping grit, Waveshape for
  even-harmonic bite. (A / B)
- **Shape** — the waveshaper curve from the suite bank: **Tube** (soft, warm), **Tape**
  (rounded, compressed), **Fold** (aggressive wavefolding), **Hard** (clipping edge).
- **Trim** — input trim into the wet path only; the dry path stays pristine so **Mix** always
  nulls cleanly. −24…+24 dB.
- **Drive** — base drive into the shaper before any sidechain modulation. The core amount of
  saturation. 0…48 dB.
- **Depth** — how much the sidechain envelope modulates the distortion; 0 % is a static
  saturator, 100 % is fully SC-driven pumping. 0…100 %.
- **Curve** — envelope shaping exponent in Mode A: below 1.0 makes the grit snap up fast on
  transients, above 1.0 makes it swell more gradually. 0.25…4.0.
- **Attack** — sidechain envelope attack; short values grab the transient, longer values ride
  the body. 0.1…200 ms.
- **Release** — sidechain envelope release; sets how long the grit hangs after each hit, the
  key to a rolling techno pump. 5…2000 ms.
- **SC Focus** — center frequency of the sidechain focus band the envelope tracks; put it on the
  kick fundamental for tight low-end pumping. 20…20000 Hz.
- **SC Width** — bandwidth of that focus band in octaves (→ bandpass Q); narrow to lock onto one
  element, wide to follow the whole sidechain. 0.2…4.0 oct.
- **SC Listen** — monitor the sidechain focus band itself, so you can hear exactly what the
  envelope is tracking while you tune Focus/Width. on/off.
- **Pre HP** — high-pass before distortion; lift it to keep sub energy out of the shaper so the
  low-end stays clean. 20…2000 Hz.
- **Pre LP** — low-pass before distortion; pull it down to feed the shaper a darker, rounder
  source. 200…20000 Hz.
- **Post HP** — high-pass after distortion; tames rumble and mud generated by the saturation.
  20…2000 Hz.
- **Post LP** — low-pass after distortion; the main tone control for taming fizz and harshness on
  the harmonics. 200…20000 Hz.
- **Auto-Gain** — matches post-distortion RMS back to the input over ~300 ms (±12 dB clamp) so
  driving harder doesn't just get louder. on/off.
- **Mix** — dry/wet blend; at 0 % the output nulls against the dry input, so use it for parallel
  grit. 0…100 %.
- **Out** — final output trim after the mix. −24…+24 dB.

## Recipes

1. **Rolling Techno Rumble** *(start: Techno Rumble Driver)* — set **Mode** = A · Env-Drive,
   **Shape** = Hard, **Drive** ≈ 30 dB, **Depth** 70 %, **SC Focus** ≈ 60 Hz, **SC Width** 1.0 oct,
   **Attack** 2 ms, **Release** 180 ms. Route the kick to the sidechain: the distortion swells
   between hits and ducks on each kick, giving that continuous warehouse rumble. Trim harshness
   with **Post LP** ≈ 6 kHz and leave **Auto-Gain** on.
2. **Vocal-Rip Crush** *(start: Vocal Crush)* — **Mode** = B · Waveshape, **Shape** = Tape,
   **Drive** ≈ 18 dB, **Depth** 55 %, key the sidechain off the vocal itself with **SC Focus**
   ≈ 1.5 kHz so consonants pump the bias. Pull **Pre HP** up to 120 Hz to keep the low mud out,
   set **Post LP** ≈ 8 kHz, **Mix** 80 % for a ruined-transmission vocal.
3. **Atmospheric DnB Drum-Bus Heat** *(start: Drum Bus Pump-Drive)* — **Mode** = A · Env-Drive,
   **Shape** = Tube, **Drive** ≈ 10 dB, **Depth** 40 %, **Attack** 5 ms, **Release** 90 ms, sidechain
   from the kick/snare bus. Keep **Mix** near 100 % with **Auto-Gain** on for glue that breathes
   with the break without obvious level jumps.
4. **Parallel Fold Texture** *(start: Pad Ring-Fold)* — **Mode** = B · Waveshape, **Shape** = Fold,
   **Drive** ≈ 24 dB, **Depth** 60 %, and drop **Mix** to ~35 % so the folded ring sits under the
   dry pad. Flip **SC Listen** on briefly to place **SC Focus** on the part of the pad you want to
   excite, then back off.

Kick Bass Grit · Vocal Crush · Pad Ring-Fold · Drum Bus Pump-Drive · Techno Rumble Driver.

## Testing in FL

1. Options → Manage plugins → "Find more plugins", then add **Qeynos GRIT** to a
   channel/mixer insert.
2. Route a kick (or drum bus) to GRIT's **sidechain** input (FL: use the plugin
   wrapper's sidechain routing on the mixer — send the kick track to the GRIT track
   and pick it as the sidechain source).
3. Start with **Kick Bass Grit**; the distortion should pump with the kick. Toggle
   **SC Listen** to hear the focus band the envelope is tracking.

Offline audition renders (1 kHz sine main + pulsed sidechain, one per preset) are in
`renders/GRIT/`.
